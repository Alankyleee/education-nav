# 关于本站

## 定位与使命

**教育学学术资源导航**是一个面向**教育研究者、教育学研究生（硕士/博士）与高校教师**的资源索引平台。

本站不托管原始文献，而是对公开可访问的学术资源进行**系统整理、分类注释与导航**，旨在降低研究者寻找优质资源的时间成本。

## 内容范围

| 模块 | 内容 |
|------|------|
| 数据库与数据 | 国内外开放教育数据集、统计年鉴、文献检索数据库 |
| 政策文献 | 国内外教育政策原文、比较教育政策分析资源 |
| 学术书单 | 教育学理论、研究方法、比较教育、教育技术书目 |
| 数据分析工具 | 统计软件、质性分析工具、文献管理、学术写作工具 |
| 视频与课程 | 研究方法讲座、名校公开课、学术会议录像 |

## 使用说明

!!! info "资源链接"
    本站所有外部链接均指向原始来源。由于网络资源可能变更，若发现失效链接，欢迎反馈。

!!! warning "版权声明"
    本站仅提供资源导航，不提供版权受保护内容的直接下载。请通过合法渠道（机构订阅、开放获取、图书馆）获取相关资源。

## 贡献与反馈

本站欢迎社区贡献：

- **推荐资源：** 如有优质学术资源希望收录，欢迎提交建议
- **纠错：** 若发现信息错误或链接失效，请告知
- **内容贡献：** 欢迎对特定领域有深入了解的研究者贡献更详细的资源说明
