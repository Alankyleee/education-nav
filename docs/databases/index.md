## 使用须知

!!! warning "数据使用规范"
    使用公开数据集开展研究时，请仔细阅读各数据库的**数据使用协议（DUA）**，在论文中正确引用数据来源，部分数据集需申请授权方可访问。

!!! info "数据格式说明"
    大多数国际数据库提供 SPSS、Stata、SAS 格式数据，推荐使用 [R 或 Python](../tools/statistics.md) 进行跨平台分析。
