## 重要国际政策框架

!!! note "教育2030框架（SDG4）"
    可持续发展目标第4项：**确保包容和公平的优质教育，让全民终身享有学习机会。**

    - 目标文件：[https://unesdoc.unesco.org/ark:/48223/pf0000245656](https://unesdoc.unesco.org/ark:/48223/pf0000245656)
    - 仁钦报告（Incheon Declaration）：2015年世界教育论坛宣言

!!! note "教师地位建议书（1966/2024）"
    UNESCO/ILO《关于教师地位的建议书》，规定教师权利、职责与工作条件的国际标准，2024年修订版。

## 政策文献检索工具

- **ERIC 政策报告检索：** 在 [https://eric.ed.gov](https://eric.ed.gov) 筛选 Publication Type → Reports（Policy）
- **OECD iLibrary：** [https://www.oecd-ilibrary.org](https://www.oecd-ilibrary.org) — 免费摘要，部分全文需订阅
- **世界银行开放知识库：** [https://openknowledge.worldbank.org](https://openknowledge.worldbank.org) — 大量报告免费获取
