### NVivo 核心工作流

```
1. 导入数据（访谈稿、文档、视频、图片、社交媒体数据）
   ↓
2. 创建节点（Nodes）进行开放编码
   ↓
3. 整合节点为主题（主轴编码）
   ↓
4. 使用查询功能（Word Frequency、Text Search、Matrix Coding）
   ↓
5. 可视化（词云、节点树、模型图）
   ↓
6. 导出编码报告与分析备忘录
```

## 编码方法论参考

!!! note "扎根理论编码"
    参考 Strauss & Corbin《扎根理论研究方法》中的三级编码：
    1. **开放编码（Open Coding）：** 逐行分析，命名概念
    2. **主轴编码（Axial Coding）：** 建立概念间关系
    3. **选择性编码（Selective Coding）：** 形成核心类属

!!! note "主题分析（Thematic Analysis）"
    Braun & Clarke（2006）六步骤框架，最常用的质性分析方法：
    1. 熟悉数据 → 2. 生成初始编码 → 3. 搜寻主题 → 4. 审查主题 → 5. 定义主题 → 6. 撰写报告

    **关键文献：** Braun, V., & Clarke, V. (2006). Using thematic analysis in psychology. *Qualitative Research in Psychology, 3*(2), 77-101.

## AI 辅助质性分析

!!! warning "使用注意"
    AI工具可辅助质性编码，但**不应替代研究者的诠释性判断**。使用AI辅助编码时需在论文中说明并评估信度。

| 工具 | 用途 |
|------|------|
| ATLAS.ti AI | 自动建议编码，情感分析 |
| ChatGPT / Claude | 辅助备忘录写作、主题命名 |
| Dovetail | 用户研究导向的质性分析平台 |
| Otter.ai / 讯飞听见 | 访谈录音自动转写 |
