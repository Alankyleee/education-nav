# 学术写作辅助工具

## 排版工具

### LaTeX

学术论文排版的黄金标准，尤其适合包含大量公式、表格的教育量化研究论文。

| 工具 | 说明 |
|------|------|
| **Overleaf** | [https://www.overleaf.com](https://www.overleaf.com) — 在线LaTeX编辑器，免费版满足多数需求，协作方便 |
| **TeXLive / MiKTeX** | 本地LaTeX发行版（Windows/Mac/Linux） |
| **VS Code + LaTeX Workshop** | 本地编辑，实时预览，推荐给有编程基础的研究者 |

**入门资源：**
- *The Not So Short Introduction to LaTeX 2e*（免费PDF，搜索 lshort-zh-cn 获取中文版）
- Overleaf 官方教程：[https://www.overleaf.com/learn](https://www.overleaf.com/learn)

### Markdown 写作

- **Obsidian：** [https://obsidian.md](https://obsidian.md) — 本地知识管理与笔记，支持文献插件
- **Typora：** 所见即所得 Markdown 编辑器
- **Quarto：** [https://quarto.org](https://quarto.org) — R/Python + Markdown，适合可重复研究报告

## AI 写作辅助

!!! warning "学术诚信声明"
    使用 AI 工具辅助写作时，请遵守所在机构的学术诚信政策。多数期刊要求**声明 AI 工具使用情况**，AI 生成内容不得直接作为论文主体提交。建议将 AI 用于：头脑风暴、提纲整理、语言润色，而非生成核心论点与实证分析。

| 工具 | 适用场景 |
|------|----------|
| **Claude** | 文献综述框架讨论、写作反馈、研究问题打磨 |
| **ChatGPT** | 写作辅助、代码生成、方法论讨论 |
| **Elicit** | [https://elicit.com](https://elicit.com) — AI文献综述辅助，自动提取论文关键信息 |
| **Consensus** | [https://consensus.app](https://consensus.app) — 基于实证研究的问答，减少幻觉 |
| **Research Rabbit** | [https://www.researchrabbit.ai](https://www.researchrabbit.ai) — 文献关系图谱与推荐 |
| **Connected Papers** | [https://www.connectedpapers.com](https://www.connectedpapers.com) — 可视化文献引用关系网络 |
